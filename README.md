# SplitPro

SplitPro is a web app that helps you split expenses with friends easily.  
It lets you create groups, add expenses, and see who owes whom.

This project was built as part of an internship assignment.

---

## Features

- Create groups and add members
- Add expenses with equal or custom splits
- See balances between members
- Settle up payments
- View spending insights with charts
- AI-based expense categorization and insights

---

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL
- Authentication: JWT
- Charts: Recharts
- AI: GROQ LLM API
